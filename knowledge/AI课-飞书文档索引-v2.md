## 效率工具安装 SOP

以下工具按需安装，老鸟强烈推荐：

1. [CCswitch 安装与使用完整指南](https://k8ncicd9x7.feishu.cn/wiki/HMduw5stQiWd8tkD7wOcB4TZni7)：快速切换不同 API，无需手动设置，老鸟必备。
2. [闪电说--安装SOP](https://k8ncicd9x7.feishu.cn/wiki/F05IwACAHi8UXskKJYIcTyHWnTc)：打字很累时的语音输入解决方案，老鸟必备。
3. [Claude Code 接入飞书 MCP 安装SOP](https://k8ncicd9x7.feishu.cn/wiki/ZtLdwoeKXiZMzpkX7svckQDYnFf)：实现 Claude Code 和飞书的共创协作，老鸟必备。
4. [爬取网站、小红书、公众号等数据-安装SOP](https://k8ncicd9x7.feishu.cn/wiki/LBwYwCj5Uiw0xek4KyfcQie5nNd)：基于 Chrome DevTools MCP + Claude Code，无需手动编写爬虫代码，老鸟必备。
5. [Python 插件安装SOP](https://k8ncicd9x7.feishu.cn/wiki/Pz0nweY3jipa6FkruLTcswPLneg)：搭建 Python 运行环境，支持数据可视化、自动化脚本、文件批量处理等高阶功能，老鸟必备。
6. 读取视频+音频文件安装SOP（链接待补充）：安装 FFmpeg 和 Whisper，支持 Claude Code 自动处理视频/音频文件及语音转文字，适合有音视频分析需求的用户。
