# 数据库与 Supabase FAQ

## 怎么给网站加登录注册功能？

使用 Supabase 来实现网站的用户登录注册功能。详细教程见课程资料，按照课程步骤操作即可。

## Supabase 注册没反应？

去 Supabase 的 Table Editor 看是否有数据：
- 有数据：说明后端已经成功，问题在前端显示，让 AI 修复前端显示部分
- 没有数据：说明后端也没有成功，需要检查 Supabase 连接配置
