# 退款政策

## 什么情况下可以申请退款

以下情况通常可以申请退款：

1. 用户重复支付。
2. 系统故障导致服务长期无法正常使用。
3. 平台确认存在异常扣费。
4. 在明确承诺的服务范围内未完成交付。

## 不支持退款的情况

以下情况通常不支持退款：

1. 服务已经正常交付并使用完成。
2. 因用户个人原因不再继续使用。
3. 非平台原因造成的支付或使用问题。
4. 超过平台规定的售后处理时效。

## 退款申请需要提供什么

请用户提供以下信息：

1. 账号 ID 或注册手机号。
2. 支付时间。
3. 支付金额。
4. 支付截图。
5. 问题描述。

## 退款处理流程

客服收到申请后，先核对订单和账号信息。
确认符合退款条件后，提交给管理员审核。
审核通过后，将按原支付路径退回。

## 退款到账时间

退款申请审核通过后，通常会在 1 到 7 个工作日内到账。
实际到账时间以支付渠道和银行处理速度为准。

## 客服回复示例

可以先把您的账号 ID、支付截图和支付时间发给我，我帮您核对订单。
如果确认符合退款条件，会尽快为您提交审核处理。
