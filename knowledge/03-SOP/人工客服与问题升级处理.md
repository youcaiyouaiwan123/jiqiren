# 人工客服与问题升级处理

## 什么情况建议转人工

以下情况建议尽快转人工处理：

1. 用户反馈涉及退款、投诉或敏感纠纷。
2. 用户账号数据异常，无法由标准流程解决。
3. 用户提供了订单、支付、封禁等复杂问题。
4. 用户多次沟通后仍未解决。

## 转人工前需要收集哪些信息

转人工前建议先收集以下信息：

1. 用户账号 ID。
2. 联系方式。
3. 问题现象描述。
4. 报错截图或相关图片。
5. 最近一次操作时间。

## 问题升级处理流程

先确认问题是否可以用已有知识库直接解决。
如果不能解决，整理用户信息和问题摘要。
随后将问题提交给管理员或技术人员继续跟进。

## 提交给技术排查时的建议格式

建议包含以下内容：

1. 用户 ID。
2. 会话 ID。
3. 问题描述。
4. 操作步骤。
5. 报错信息。
6. 发生时间。

## 客服回复示例

这个问题需要进一步核实，我先帮您记录并提交人工处理。
您可以把账号信息和相关截图发给我，我们会尽快跟进。
