# 安装配置 FAQ

## 开课前需要安装哪些工具？总费用多少？

开课前需要准备以下工具，总费用约 220 元/月：

费用明细：
- 科学上网工具：18 元/月（Clash 推荐）
- Claude Pro 账号：150 元/月（淘宝买成品独享号）
- Cursor/VS Code：免费
- Claude Code：API 按量计费，先充 50 元
- Obsidian：免费

安装顺序建议：
1. 先装科学上网（后面所有工具都要用外网）
2. 装 Claude Pro 网页版（先能用起来聊天）
3. 装 Node.js + Git（系统必备，后面装 Claude Code 要用）
4. 装 Cursor 或 VS Code（编辑器二选一，推荐 Cursor）
5. 在编辑器终端里装 Claude Code
6. 装 Obsidian（知识管理）
7. 装 CC Switch（API 一键切换工具，选装）
8. 装闪电说（语音输入工具，选装）

每个工具都有详细的安装教程文档，遇到问题可以约郭老师/天蓝老师/杏杏老师远程协助。
远程协助工具用「连连控」：asklink.com

## 科学上网工具怎么安装？推荐哪个？

推荐用 Clash，18 元/月，性价比最高。

Clash 安装步骤：
1. 注册购买：访问 dash.watchdoor.dog/login，注册后选流量套餐
2. 获取订阅：后台选对应系统（Windows/Mac），复制订阅链接
3. 导入订阅：打开 Clash 软件 → 进入「订阅管理」→ 粘贴链接 → 点击「导入」
4. 启用代理：开启「系统代理」，在「节点选择/AI平台/GLOBAL」三个入口都选同一个海外节点（如美国1、日本1）
5. 打开/关闭：用外网时手动开启「系统代理」+「虚拟网卡」，不用时点击关闭（变灰色）

其他选项：
- Fic：25 元/月，功能和 Clash 完全一致，Clash 装不了可以用这个，找老师要安装包
- v2rayN：50 元/月，网速更快更稳定，找老师要

超级重要：节点必须选纯海外（美国、日本等），严禁选香港/台湾节点！香港台湾算中国境内，会导致 Claude 封号！

## 科学上网有什么注意事项？怎么防止 Claude 被封号？

几个关键注意事项，务必记住：

1. 节点选择：只选纯海外节点（美国、日本等），严禁选香港/台湾（算中国境内会封号）
2. 别频繁换节点：选定一个后尽量不换，单日最多换 1 次，换了之后等半天再用
3. 纯净度检测：每次开科学上网后，先访问 ping0.cc 测试，数值过高说明节点被污染了，换一个或者少用 Claude
4. 打开/关闭顺序（重要）：
   - 开的时候：先开科学上网 → 测纯净度 → 再打开 Claude
   - 关的时候：先关 Claude 网页 → 再关科学上网
5. 用完必须关：不用外网时手动关掉系统代理和虚拟网卡，禁止后台一直挂着
6. 新手前几天轻量使用，别一上来就疯狂对话，有个熟悉的过程

万一被封号了：联系淘宝店铺售后，成品独享号一般有售后保障，可以换号。

## Claude Pro 网页版怎么登录？

购买账号：
- 淘宝搜「Claude Pro 成品号」，150-160 元/月（180 元/月售后更好）
- 确认店铺有「售后保障 + 快速发货 + 独享账号」
- 购买后找客服要账号信息，格式：邮箱----密码----令牌
- 令牌暂时不用，妥善保存

登录步骤：
1. 先开启科学上网，打开 ping0.cc 测纯净度
2. 谷歌浏览器打开 claude.ai/login
3. 新开标签页打开 mail.com，用买到的邮箱+密码登录邮箱
4. 回到 Claude 登录页，输入对应邮箱地址，点 Continue
5. 切回 mail.com 邮箱，找到 Anthropic 官方发的登录链接邮件，点击邮件里的登录按钮
6. 自动跳转到 Claude 使用界面，登录完成！

以后每次用：开科学上网 → 浏览器输入 claude.ai → 直接用

注意：账号是独享的，不要分享给别人，会封号；用完先退出 Claude 网页，再关科学上网。

## Cursor 怎么安装和配置？

Cursor 是 AI 编程首选编辑器，免费，操作比 VS Code 更简洁。

安装步骤：
1. 如果官网打不开，先开科学上网
2. 访问 cursor.com/cn，选对应系统版本下载安装
3. 打开后点 SIGN UP 注册：
   - 点 Continue → 输入国内邮箱（163/QQ 都行）→ 获取验证码
   - 填姓名（首字母必须大写，硬性要求）
   - 输入手机号（格式：86138XXXXXXXX，前面加 86）→ 获取手机验证码
   - 点 Yes Login → 浏览器弹出授权窗口 → 点 Yes, Log In
4. 返回 Cursor，连续两次点 Continue 完成初始化（默认配置就行）
5. 在电脑上新建一个「AI开发」文件夹，在 Cursor 点 Open 导入

重要提示：
- 不需要开通付费 Pro 版，免费功能完全够用
- 姓名和手机号格式严格按要求填，否则注册不了

## Node.js 和 Git 怎么安装？

这两个是系统必备的基础工具，安装 Claude Code 之前必须先装好。

下载地址：
- Node.js 官网：nodejs.org（选 LTS 长期支持版）
- Git：课程群里有安装包，也可以去 git-scm.com 下载

Windows 系统：
- 下载 `node-windows.msi` 和 `Git-windows.exe`
- 双击运行，一路点 Next/下一步就行

Mac 系统：
- 下载对应的 `.pkg` 文件
- 双击安装，按提示操作

验证安装成功：
- 打开终端/命令行
- 输入 `node --version` 看到版本号就 OK
- 输入 `git --version` 看到版本号就 OK

装完这两个，就可以在编辑器（Cursor/VS Code）的终端里安装 Claude Code 了。

## 闪电说是什么？怎么安装使用？

闪电说是一款语音输入 + AI 交互工具，可以用说话代替打字来操作 AI，特别适合不想打字的场景。

费用：免费！官方赠送 500 万 tokens。

安装步骤：
1. 访问 shandianshuo.cn，选对应系统版本下载
2. 按安装向导一路下一步
3. 安装完会引导你测试快捷键：按键盘空格键右边的 Alt 键
4. 测试通过就进入主界面了

配置 API（用火山引擎的免费额度）：
- 打开闪电说 → 主页 → 点「查看完整教程」
- 按教程去火山引擎官网注册，获取 API Key
- 回闪电说填入 API 配置
- 官方赠送 500 万 tokens，够用很久

主界面 6 个模块：主页、记忆、技能、模型、设置、用户中心。有问题找老师帮忙配置。

## Excel 怎么接入 Claude？

可以在微软原生 Excel 里直接调用 Claude，让 AI 帮你处理表格数据。

必备条件：
- Claude Pro 账号（必须）
- 科学上网工具（全程开启）
- 微软 Office 365 账号（闲鱼约 5 元可买）
- 系统原生 Excel（不支持 WPS！）

安装步骤：
1. 开启科学上网 + 登录 Microsoft Office 365 账号
2. 打开 Excel，点右上角「加载项」
3. 搜索 `claude in excel` → 点安装 → 安装完关闭 Excel
4. 重新打开 Excel，点右上角 Claude 图标 → 点 Login
5. 输入 Claude Pro 邮箱 → 点继续 → 会进入验证码页面，先别动
6. 新开浏览器打开 mail.com → 登录 Claude 邮箱
7. 找到 Anthropic 发的验证码邮件 → 点黑色模块复制验证码
8. 回到 Excel 粘贴验证码 → 点确定 → 点授权
9. 关闭 Excel 重新打开，检查右侧 Claude 面板是否登录成功
10. 如果还显示 Login，重复 4-8 步（网络波动可能要多试几次）

注意：只支持微软原生 Excel，WPS 不行！

## 飞书MCP配置时，授权链接打不开（localhost:3000 无法访问）怎么办？

原因：MCP 服务没有正常启动。

解决方法：
1. 确保已安装 Node.js
2. 重启 Claude Code
3. 检查配置文件格式是否正确（JSON 格式要求严格）

## 飞书MCP提示 Token 过期怎么处理？

原因：授权已过期，需要重新授权。

解决方法：
1. 在 Claude Code 中触发任意飞书操作
2. 系统会生成新的授权链接
3. 在浏览器中打开链接完成授权

## 飞书MCP提示权限不足怎么解决？

原因：应用没有开通对应的 API 权限。

解决方法：
1. 回到飞书开放平台
2. 进入应用的「权限管理」→「API 权限」
3. 开通需要的权限
4. 重新发布应用版本

## 飞书MCP更换了 App Secret 后连接失败怎么办？

原因：配置文件中的 Secret 没有同步更新。

解决方法：
1. 打开 `.claude.json` 配置文件
2. 找到 `mcpServers → lark-mcp → args` 中 `-s` 参数后面的值
3. 替换为新的 App Secret
4. 保存文件并重启 Claude Code
